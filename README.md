## ✅ 참여 방법
1. Repository `fork`
2. 코딩 테스트 문제를 풀고 GitHub에서 Pull Request 생성
3. 본인의 알고리즘 레포지토리에 코드 푸시
4. PR 생성 후 링크 복사
5. 링크 형식: `https://github.com/{owner}/{repo}/pull/{number}`
6. `/코테제출 https://github.com/username/algorithm/pull/123`
7. `Contribute`로 `Organization`에도 밀어넣는다-!
